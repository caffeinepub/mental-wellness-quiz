# Specification

## Summary
**Goal:** Fix the quiz question progression issue preventing users from advancing through questions.

**Planned changes:**
- Debug and fix the button/answer submission logic that blocks question progression
- Verify quiz submission data flow works correctly for both authenticated (backend) and anonymous (localStorage) users
- Ensure all quiz questions can be completed and results page is reached

**User-visible outcome:** Users can successfully select answers, progress through all quiz questions, and view their results without errors.
