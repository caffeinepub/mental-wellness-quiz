# Specification

## Summary
**Goal:** Add user authentication with Internet Identity and enable users to track their quiz history over time.

**Planned changes:**
- Integrate Internet Identity authentication for secure sign-up and login
- Associate quiz submissions with authenticated user principals in the backend
- Create a user profile/history page displaying past quiz results and progress trends
- Add backend functions to retrieve and store user-specific quiz history
- Update quiz submission flow to save results to authenticated user accounts while maintaining anonymous quiz support

**User-visible outcome:** Users can sign up and log in using Internet Identity, take quizzes with results saved to their account, and view their mental wellness journey over time through a personalized history page. Anonymous users can still take quizzes without authentication.
