# Specification

## Summary
**Goal:** Add a sign up option to the authentication flow so new users can create accounts using Internet Identity.

**Planned changes:**
- Add a visible sign up button or link for unauthenticated users
- Update the AuthButton component to differentiate between sign in and sign up actions
- Add sign up option to prominent locations like HomePage and ResultsPage
- Ensure sign up initiates the Internet Identity authentication flow

**User-visible outcome:** Users will see a clear sign up option that allows them to create a new account using Internet Identity, distinct from the sign in option for existing users.
